# Google AI Agents Course Notes

A comprehensive student-created reference guide for Google's **5-Day AI Agents: Intensive Vibe Coding Course** conducted through Kaggle.

---

## About This Repository

This repository contains detailed notes, explanations, architecture diagrams, flowcharts, codelab walkthroughs, interview preparation material, and practical examples based on the complete Google AI Agents course.

The goal is not only to summarize the course, but also to provide a beginner-friendly reference guide that helps students understand modern AI Agent development from fundamentals to production-grade systems.

This repository is designed for:

* Students
* Beginners
* Software Developers
* AI Enthusiasts
* Hackathon Participants
* Open Source Contributors

---

## Learning Objectives

After studying this repository, readers should understand:

* What AI Agents are and how they differ from traditional LLMs
* Agentic Engineering and Vibe Coding
* Context Engineering and Context Rot
* Model Context Protocol (MCP)
* Agent Skills and Procedural Memory
* Google Antigravity IDE
* Agent Development Kit (ADK 2.0)
* Human-in-the-Loop Workflows
* Ambient Agents
* Agent Security and Evaluation
* STRIDE Threat Modeling
* Spec-Driven Development
* Production-Grade Agent Architectures
